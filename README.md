URL Shortener using MongoDB, Express JS and NodeJS. Designed in MVC pattern.

*NOTE*: Fronted is not developed for this project yet. You can test out the backend server in Postman.

**1.)** POST /url : for creating the short url. 
**2.)** GET  /url/analytics/:shortId : to see the analytics of the short url.
**3.)** GET  /:shortId : to redirect to the original url

Run "npm install" to install the dependencies.
